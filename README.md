# 基于Django的Python机票预订系统 python757

## 项目概述

本项目是基于Python语言的Django框架开发的一款机票预订系统，包含了用户端和管理页两大功能模块，旨在提供便捷的航班预订、管理以及相关服务。

## 技术栈

- **开发语言**：Python（建议版本 3.11）
- **开发框架**：Django
- **开发工具**：Pycharm
- **数据库**：Mysql
- **数据库软件**：Navicat/Dbeaver

## 功能模块

### 用户端功能

- 机票查询
- 机票预订
- 机票退订
- 订单展示
- 登录注册
- 机票营收统计

### 管理页功能

- 营收管理
- 订单管理
- 航班管理

## 系统角色

- 用户：可以进行机票查询、预订、退订等操作
- 管理员：负责营收管理、订单管理、航班管理等功能

## 运行环境

- Python 3.11 或以上版本
- Mysql 数据库
- 支持Django的Web服务器

## 部署步骤

1. 确保Python环境和Mysql数据库已正确安装和配置。
2. 克隆或下载项目代码到本地。
3. 使用Pycharm等工具打开项目并配置数据库连接。
4. 运行Django项目。
